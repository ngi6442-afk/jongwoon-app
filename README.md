# 종운 업무체크 (jongwoon-app)

JW-05-011 주기별 업무체크리스트를 홈 화면 앱처럼 사용할 수 있는 설치형 PWA입니다.

## 설치 순서

1. GitHub 저장소 **Settings → Pages → Build and deployment → Deploy from a branch** 에서 `main` / `/(root)` 를 선택하고 저장합니다.
2. 1~2분 정도 기다린 뒤 다음 주소로 접속합니다.
   `https://ngi6442-afk.github.io/jongwoon-app/`
3. 최초 접속 시 표시되는 화면에서 관리자에게 받은 앱 토큰을 입력합니다.
4. 브라우저 메뉴에서 "홈 화면에 추가"를 선택하면 아이콘이 생성되어 일반 앱처럼 사용할 수 있습니다.

## 동작 방식

- 체크 항목을 누르면 오늘 날짜가 기록되고, 다시 누르면 해제됩니다.
- 주간 / 월간 / 분기 / 반기 / 연간 주기가 바뀌면 해당 항목은 자동으로 초기화되며, 과거 기록은 보존되어 미체크 항목에 "지난 확인 M/D"로 표시됩니다.
- 데이터는 비공개 저장소(`jongwoon-appdata`)의 `data/checklist.json`에 저장되며, 기기 간에도 동일하게 동기화됩니다.
- 네트워크가 끊기면 화면 상단에 "오프라인 — 연결되면 자동 저장" 안내가 표시되고, 마지막으로 불러온 기록을 그대로 보여줍니다. 연결이 복구되면 자동으로 저장을 재시도합니다.
- 설정(⚙) 버튼으로 앱 토큰을 다시 입력할 수 있습니다.

## 파일 구성

- `index.html` — 앱 본체 (단일 파일, 외부 CDN 미사용)
- `manifest.webmanifest` — PWA 설치 메타데이터
- `sw.js` — 앱 셸 캐시용 서비스워커 (데이터 API는 네트워크 전용)
- `icon-192.png`, `icon-512.png` — 앱 아이콘
